# Freedeck - Steam Deck 游戏管理插件

本软件仅限研发测试，禁止商业获利。资源源自互联网，仅用于功能测试。请在使用后 24 小时内删除

 
本项目为 Steam Deck 专用插件，提供游戏下载、游戏库管理、云存档同步及一键安装功能。
功能
- 不限速游戏下载​
- 游戏库资源检索与展示​
- 跨设备云存档同步​
- 一键安装与运行，无需额外配置

适用平台 
- Steam Deck 掌机​
- SteamOS 系统

安装 
- 进入 Steam Deck 桌面模式​
- 安装 Decky Loader
- 通过github releases安装本插件
- 重启插件加载器生效
